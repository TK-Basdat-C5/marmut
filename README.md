Tips for run:
1. Make env 
```
python -m venv env
```

2. Masuk env
```
env\Scripts\activate.bat
```

3. Install requirement
```
pip install -r requirements.txt
```

4. Run program
```
py manage.py runserver
```

5. Make .env file in root folder, then insert DB_USER and DB_PASSWORD
```
DB_USER=[user DB]
DB_PASSWORD=[password DB]
```
